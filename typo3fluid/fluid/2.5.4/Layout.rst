======
layout
======


With this tag, you can select a layout to be used for the current template.

= Examples =

<code>
<f:layout name="main" />
</code>
<output>
(no output)
</output>

Arguments
=========


name (string)
-------------


Name of layout to use. If none given, &quot;Default&quot; is used.
